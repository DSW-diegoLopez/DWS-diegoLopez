Flex Box

El Módulo de Caja Flexible, comúnmente llamado flexbox, fue diseñado como un modelo unidimensional
de layout, y como un método que pueda ayudar a distribuir el espacio
entre los ítems de una interfaz y mejorar las capacidades de alineación
Developer.mozilla.org. 2022. Conceptos Básicos de flexbox - CSS | MDN. [online] Available at: https://developer.mozilla.org/es/docs/Web/CSS/CSS_Flexible_Box_Layout/Basic_Concepts_of_Flexbox [Accessed 30 June 2022].
Listas

Existen tres tipos de listas en HTML que nos permiten organizar la información: listas numeradas, listas desordenadas y las listas de definición.
W3schools.com. 2022. HTML Lists. [online] Available at: https://www.w3schools.com/HTML/html_lists.asp [Accessed 30 June 2022].
enlaces en HTML

El Elemento HTML Anchor <a> crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL
