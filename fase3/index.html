Contenido Explicativos Html 5

Aqui encontrars contenidos explicativos que te permita utilizar Flex Box, Listas y enlaces en HTML

Flex Box

El Módulo de Caja Flexible, comúnmente llamado flexbox. Fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación

Listas

Existen tres tipos de listas en HTML que nos permiten organizar la información: listas numeradas, listas desordenadas y las listas de definición.

Enlaces

El Elemento HTML Anchor <a> crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL
